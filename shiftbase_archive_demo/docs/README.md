## 기준 문장

Shiftbase는 동영상 강의가 아닌 실습형 코스로 학습하는 AI 교육 플랫폼입니다. 사용자는 강의를 보기만 하지 않고, AI가 안내하는 단계형 화면에서 내용을 입력, 선택, 판단하며 여러 가지 AI 활용 상황을 직접 수행합니다.

## 현재 문서

- [product-definition.md](./product-definition.md): Shiftbase가 무엇인지, 무엇이 아닌지, 어떤 표현을 써야 하는지 정리한다.
- [course-experience-model.md](./course-experience-model.md): 실습형 코스의 화면 구조, 코스 단위, 단계 구성, 블록 기준을 정리한다.
- [technical-operations.md](./technical-operations.md): 현재 구현 상태와 앞으로의 운영 구조를 정리한다.

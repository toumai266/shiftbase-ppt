# 법무와 정책 체크리스트

## 목적

이 문서는 Shiftbase 출시 전 확인해야 할 기본 법무와 정책 항목을 정리합니다. 법률 자문을 대체하지 않습니다.

## 브랜드와 상표

확인 항목:

- Shiftbase 상표 사용 가능성
- 유사 상표 존재 여부
- 도메인 사용 가능성
- 로고 사용 가능성
- 영문과 한글 표기 정책

권장:

- 정식 출시 전 상표 검색
- 브랜드명 후보 예비안 확보
- 도메인과 SNS 핸들 확보

## 이용약관

포함 항목:

- 서비스 목적
- 회원가입과 계정 관리
- 유료 서비스와 환불
- 학습 콘텐츠 이용 범위
- 템플릿과 자료의 사용 범위
- 금지 행위
- 서비스 변경과 중단
- 책임 제한

## 개인정보 처리방침

포함 항목:

- 수집하는 개인정보
- 수집 목적
- 보관 기간
- 제3자 제공 여부
- 처리 위탁 여부
- 사용자 권리
- 문의처

주의:

- 실제 고객정보 입력은 권장하지 않는다는 안내 필요
- AI 호출 시 외부 모델 제공자에게 데이터가 전송될 수 있는지 고지 필요

## AI 사용 고지

필수 고지:

- AI가 생성한 결과는 업무 보조용 초안입니다.
- 사용자는 발송 전 직접 검토해야 합니다.
- 민감정보를 입력하지 말아야 합니다.
- 업종별 법적 책임이 있는 안내는 전문가 검토가 필요합니다.

## 환불 정책

정해야 할 항목:

- 단일 모듈 구매 후 환불 기준
- 패키지 구매 후 환불 기준
- 기업교육 계약 취소 기준
- 자료 다운로드 후 환불 제한 여부
- 결제 오류 처리

## 콘텐츠 저작권

정해야 할 항목:

- 플랫폼 콘텐츠 소유권
- 템플릿 사용 범위
- 재배포 금지 여부
- 기업교육 커스텀 콘텐츠 소유권
- 사용자가 만든 결과물의 권리

권장 방향:

- 사용자가 실습 중 작성한 결과물은 사용자가 업무에 사용할 수 있게 허용
- 플랫폼 템플릿은 개인 또는 조직 내 사용 가능
- 재판매와 재배포는 제한

## 업종별 주의사항

### 의료와 클리닉

- 진단, 치료, 처방 관련 답변 제한
- 일반 예약과 운영 안내 중심으로 설계

### 금융과 보험

- 투자, 대출, 보험 판단 조언 제한
- 일반 문의 분류와 문서 정리 중심으로 설계

### 법률

- 법률 판단과 자문 표현 제한
- 일반 문의 정리와 문서 초안 보조로 제한

### 교육

- 학생 개인정보와 성적 정보 입력 주의
- 상담 기록 보관 정책 필요

## B2B 계약 체크리스트

- 교육 범위
- 수강 인원
- 제공 모듈
- 커스텀 여부
- 데이터 처리 범위
- 리포트 제공 범위
- 수료 기준
- 계약 기간
- 환불과 취소
- 보안 요구사항

## 출시 전 필수 체크

- 이용약관 작성
- 개인정보 처리방침 작성
- AI 결과 고지 작성
- 민감정보 입력 경고 작성
- 환불 정책 작성
- 템플릿 사용 범위 작성
- 상표와 도메인 검토
- 결제사 심사 준비

## 핵심 판단

Shiftbase는 업무 데이터를 다루는 학습 제품입니다. 제품 신뢰를 위해 법무와 개인정보 안내를 초기부터 정돈해야 합니다.

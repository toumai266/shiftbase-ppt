# MVP 모듈 상세 설계

## 모듈명

고객문의 자동응답과 상담 분류

## 카테고리

소규모 업체를 위한 AX

## 모듈 설명

카카오톡, 홈페이지, 인스타그램, 전화 메모 등 여러 채널에서 들어오는 고객 문의를 빠르게 정리하고 상황에 맞는 답변을 만드는 실무형 AX 학습 모듈입니다.

## 학습 대상

- 카페 운영자
- 미용실 운영자
- 학원 운영자
- 클리닉 운영 담당자
- 온라인 판매자
- 지역 서비스 업체 운영자
- 고객응대를 담당하는 작은 팀

## 학습 목표

학습자는 이 모듈을 완료한 뒤 다음을 할 수 있어야 합니다.

- 고객문의 유형을 분류할 수 있음
- 문의별 우선순위를 정할 수 있음
- 필요한 정보를 파악할 수 있음
- 상황별 답변 기준을 만들 수 있음
- AI 답변 초안을 검토하고 수정할 수 있음
- 상담 기록을 업무 자산으로 남길 수 있음

## 실습 결과물

사용자가 가져가는 결과물:

- 문의 유형 기준표
- 답변 템플릿 세트
- FAQ 템플릿
- 상담 기록 양식
- 응답 품질 체크리스트
- 실습 결과 리포트

## 모듈 구성

### 모듈 1: 문의 흐름 이해와 문제 정의

목표:

- 우리 업체에 어떤 문의가 반복되는지 파악
- 문의 채널과 응답 흐름 정리

실습:

- 샘플 문의함 확인
- 자주 반복되는 문의 표시
- 업무 문제 선택

완료 조건:

- 반복 문의 3개 이상 선택
- 개선 목표 1개 작성

### 모듈 2: 문의 유형 설계와 태깅 기준

목표:

- 문의 유형을 구분
- 분류 기준 만들기

실습:

- 문의 예시 분류
- 예약, 가격 문의, 운영 안내, 불만, 기타 선택
- 유형별 기준표 작성

완료 조건:

- 샘플 문의 5개 이상 분류
- 유형별 기준 작성

### 모듈 3: 답변 템플릿과 프롬프트 설계

목표:

- 상황별 답변 템플릿 만들기
- AI에게 답변 초안을 요청하는 방식 이해

실습:

- 답변 톤 선택
- 필수 정보 체크
- 답변 초안 생성
- 사용자 수정

완료 조건:

- 템플릿 3개 이상 저장
- 체크리스트 통과

### 모듈 4: 예약과 상담 워크플로우 연결

목표:

- 문의를 일정, 상담 기록, 후속 안내로 연결

실습:

- 예약 가능 여부 확인
- 필요한 정보 보강
- 예약 확인 메시지 작성
- 상담 기록 저장

완료 조건:

- 예약 응답 1개 완성
- 상담 기록 저장

### 모듈 5: 운영 적용과 품질 검수

목표:

- 직원이 함께 사용할 수 있는 기준 만들기
- 응답 품질을 점검

실습:

- 응답 품질 체크리스트 통과
- FAQ 템플릿 보강
- 현업 적용 계획 작성

완료 조건:

- FAQ 항목 5개 이상 작성
- 최종 리포트 생성

## 실습실 화면 구성

### 좌측 패널

- 실습 단계
- 완료 상태
- 진행 중 단계
- 실습 팁

### 중앙 패널

- 문의 목록
- 채널 필터
- 선택한 문의 상세
- AI 분석과 추천
- 답변 초안
- 액션 버튼

### 우측 패널

- 응답 품질 지표
- 문의 유형 분포
- 체크리스트
- 다음 단계 안내

### 하단 패널

- 워크플로우 한눈에 보기
- 실습 자료와 도구
- 결과와 리포트

## 샘플 문의 데이터

문의 1:

- 채널: 카카오톡
- 내용: 안녕하세요. 이번 주 토요일 2시에 2명 예약 가능한가요? 가격도 궁금합니다.
- 예상 유형: 예약, 가격 문의
- 우선순위: 보통

문의 2:

- 채널: 인스타그램 DM
- 내용: 펌 시술 가격이 어떻게 되나요? 그리고 예약은 어떻게 하나요?
- 예상 유형: 가격 문의, 예약
- 우선순위: 보통

문의 3:

- 채널: 홈페이지 문의
- 내용: 주차는 가능한가요? 주차 시간 제한이 있나요?
- 예상 유형: 운영 안내
- 우선순위: 낮음

문의 4:

- 채널: 카카오톡
- 내용: 예약 변경하고 싶어요. 5월 20일에서 5월 22일로 가능할까요?
- 예상 유형: 예약 변경
- 우선순위: 보통

문의 5:

- 채널: 전화 메모
- 내용: 학생 상담 문의드립니다. 레벨 테스트는 어떻게 진행되나요?
- 예상 유형: 상담 문의
- 우선순위: 보통

## 검증 로직

### 문의 분류

사용자가 선택한 유형이 예상 유형과 일치하면 통과합니다.

부분 일치도 허용합니다.

예시:

- 예약, 가격 문의 둘 중 하나만 선택하면 부분 통과
- 둘 다 선택하면 완전 통과

### 답변 초안

체크 항목:

- 친절한 톤
- 핵심 정보 포함
- 다음 행동 제시
- 민감정보 제외
- 과장 표현 없음

### 상담 기록

필수 항목:

- 채널
- 문의 유형
- 고객 요청
- 응답 내용
- 후속 조치

## AI 기능 적용 지점

초기 AI 기능은 다음에만 적용합니다.

- 문의 유형 자동 분류
- 답변 초안 제안
- 응답 품질 평가
- 누락 정보 확인

AI 없이도 실습이 가능하도록 샘플 결과를 내장합니다.

## MVP 성공 기준

- 사용자가 실습 1개를 끝까지 완료
- 결과 리포트 확인
- 템플릿 다운로드
- 다음 모듈에 관심 표시
- 실제 업무에 적용해볼 의향 응답

## 후속 확장

이 모듈의 구조는 다음 모듈에 재사용할 수 있습니다.

- 예약 문의를 일정으로 바꾸기
- 리뷰 분석과 답변 자동화
- 자동화된 상담 기록 관리
- 사장님을 위한 주간 운영 보고서 만들기

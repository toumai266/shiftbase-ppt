# 타깃 고객과 시장 해석

## 시장 해석

AI 교육 시장은 넓지만 이미 혼잡합니다. 단순 프롬프트 강의, 영상 강의, 도구 사용법 강의로는 차별화가 어렵습니다.

Shiftbase는 AI 교육 시장의 정면에 서기보다, 실무 교육과 업무전환 사이의 틈을 공략합니다.

핵심 시장은 다음과 같습니다.

- 소규모 업체 운영 교육
- 업무 자동화 교육
- AI 실무 도입 교육
- 기업교육과 직무 전환 교육
- 현업 적용형 디지털 전환 교육

## 시장의 미충족 요구

사용자들은 AI를 배우고 나서 다음 질문에 막힙니다.

- 내 업무 중 무엇부터 바꿀 수 있나
- 어떤 입력값을 모아야 하나
- AI에게 무엇을 맡기고 무엇은 사람이 검토해야 하나
- 결과가 괜찮은지 어떻게 판단하나
- 우리 팀이 같은 기준으로 쓰게 하려면 어떻게 해야 하나
- 교육 이후 실제 업무 산출물은 무엇인가

Shiftbase는 이 질문에 답해야 합니다.

## 1차 페르소나: 소규모 업체 운영자

### 프로필

- 카페, 미용실, 학원, 클리닉, 온라인 판매자, 지역 서비스 업체 운영자
- 직원 수 1명에서 10명
- 고객응대와 운영 업무를 직접 처리
- 기술 도입에 관심은 있으나 개발 지식은 낮음

### 주요 업무

- 카카오톡 문의 응답
- 인스타그램 DM 응답
- 예약 확정과 변경
- 리뷰 답변
- 가격 안내
- 상담 기록 정리
- 주간 운영 보고
- 안내문 작성

### 통증

- 반복 문의가 많음
- 응대 품질이 들쑥날쑥함
- 직원마다 답변 방식이 다름
- 예약과 문의가 섞여 누락됨
- 리뷰와 홍보를 꾸준히 관리하기 어려움
- AI를 써보고 싶지만 무엇부터 해야 할지 모름

### 구매 동기

- 업무 시간이 줄어드는지
- 직원 교육에 쓸 수 있는지
- 바로 가져다 쓸 템플릿이 있는지
- 복잡하지 않은지
- 비용이 부담스럽지 않은지

## 2차 페르소나: 작은 팀의 사무 실무자

### 프로필

- 스타트업, 소기업, 기관, 학원, 협회, 단체의 실무자
- 문서, 보고, 고객응대, 일정, 자료 정리를 반복함

### 통증

- 매주 비슷한 보고서를 작성함
- 회의록과 후속 업무 정리가 귀찮음
- 이메일과 문의 응답에 시간이 많이 듦
- 업무 자동화는 배우고 싶지만 개발은 부담스러움

### 필요한 모듈

- 회의록에서 보고서 만들기
- 이메일 답변 템플릿 만들기
- 주간업무 보고 자동화
- 자료 요약과 액션 아이템 정리
- 고객문의 분류와 답변 초안

## 3차 페르소나: 1인 창업자와 예비창업자

### 프로필

- 아이디어는 있으나 문서화와 실행 체계가 약함
- 사업계획서, 제안서, 홍보글, 랜딩 페이지 카피가 필요함

### 통증

- 생각은 많지만 정리하기 어려움
- 지원사업 문항 작성이 부담됨
- 경쟁사와 고객 문제를 구조화하기 어려움
- 결과물 품질을 스스로 검토하기 어려움

### 필요한 모듈

- 아이디어 메모를 사업계획서 초안으로 바꾸기
- 고객 문제 정의 실습
- 경쟁 서비스 분석 실습
- 제안서와 소개서 초안 만들기
- 발표자료 흐름 만들기

## 4차 페르소나: 기업교육 담당자

### 프로필

- 중소기업, 공공기관, 교육기관의 교육 담당자
- 직원에게 AI 활용 교육을 제공해야 함
- 단순 강의보다 결과와 리포트를 원함

### 통증

- 직원들이 교육을 들어도 업무 적용으로 이어지지 않음
- 교육 효과를 측정하기 어려움
- 부서마다 업무가 달라 공통 강의가 잘 맞지 않음
- 개인정보와 보안 이슈가 걱정됨

### 필요한 기능

- 기업별 수강자 관리
- 진도 리포트
- 실습 결과 요약
- 수료증
- 부서별 커스텀 모듈
- AI 보안 체크리스트

## 경쟁 유형

### 일반 온라인 강의 플랫폼

장점:

- 많은 강의와 강사
- 결제와 수강 기능 완비
- 대중 인지도

약점:

- 영상 중심
- 실습 결과물이 약함
- 업무 인터페이스 기반 학습이 어려움

Shiftbase의 대응:

- 영상이 아니라 실습 인터페이스를 강조
- 결과물과 현업 적용 리포트 제공

### AI 프롬프트 강의와 전자책

장점:

- 저렴함
- 빠르게 소비 가능
- 주제별 확산 쉬움

약점:

- 실제 업무흐름으로 연결이 약함
- 검토 기준과 실행 단계가 부족함

Shiftbase의 대응:

- 프롬프트보다 업무흐름 중심
- 실습과 템플릿을 결합

### 업무 자동화 SaaS

장점:

- 실제 업무 효율화 가능
- 고객에게 명확한 기능 제공

약점:

- 특정 기능에 갇힘
- 교육과 전환 경험은 약함
- 사용자가 왜 그렇게 해야 하는지 학습하기 어려움

Shiftbase의 대응:

- 완성 SaaS가 아니라 학습형 워크벤치로 포지셔닝
- 다양한 업무에 적용 가능한 사고 방식 제공

### 컨설팅과 기업교육

장점:

- 맞춤형 제공
- 신뢰와 전문성

약점:

- 비용이 큼
- 작은 업체는 접근이 어려움
- 반복 확장이 어려움

Shiftbase의 대응:

- 저가형 실습 모듈로 진입
- 반응 좋은 분야는 B2B 워크숍으로 확장

## 초기 타깃 우선순위

1. 소규모 업체 운영자
2. 작은 팀 사무 실무자
3. 1인 창업자
4. 기업교육 담당자

## 초기 메시지 우선순위

소규모 업체 대상:

- 고객문의와 예약 응대를 AI로 정리하세요.
- 반복 문의를 분류하고, 상황별 답변 기준을 만드세요.
- 직원이 바뀌어도 같은 품질로 응대할 수 있게 만드세요.

작은 팀 대상:

- 반복 문서와 보고를 AI 업무흐름으로 바꾸세요.
- 회의록, 이메일, 보고서 업무를 구조화하세요.

기업교육 대상:

- 직원들이 실제 업무 화면을 따라 하며 AI 활용을 익힙니다.
- 진도와 실습 결과를 통해 교육 효과를 확인할 수 있습니다.

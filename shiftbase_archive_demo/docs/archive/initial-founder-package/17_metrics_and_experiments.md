# 지표와 실험 설계

## 북극성 지표

초기 북극성 지표는 실습 완료 후 업무 적용 가능한 결과물을 저장한 사용자 수입니다.

단순 방문자 수나 회원가입 수보다 다음이 중요합니다.

- 실습을 시작했는가
- 끝까지 완료했는가
- 결과물을 저장했는가
- 템플릿을 다운로드했는가
- 다음 업무에도 적용하려는가

## 퍼널 지표

### 방문 퍼널

- 홈 방문
- 카테고리 클릭
- 모듈 상세 진입
- 실습 시작
- 회원가입
- 실습 완료
- 템플릿 다운로드
- 다음 모듈 이동

### 실습 퍼널

- 1단계 시작
- 2단계 완료
- 3단계 완료
- 답변 초안 생성
- 체크리스트 통과
- 결과 저장
- 리포트 확인
- 자료 다운로드

## 제품 지표

- 무료 실습 시작률
- 실습 완료율
- 평균 실습 시간
- 단계별 이탈률
- 힌트 사용률
- AI 결과 수정률
- 체크리스트 통과율
- 결과 리포트 저장률
- 템플릿 다운로드율

## 학습 효과 지표

- 사용자가 문의 유형을 정확히 분류하는 비율
- 답변 초안의 체크리스트 통과율
- 실습 전후 자신감 변화
- 현업 적용 의향
- 실제 적용 후기 수

## 사업 지표

- 이메일 수집 수
- 파일럿 신청 수
- 유료 관심 응답 수
- 구매 전환율
- 평균 결제액
- B2B 문의 수
- 파일럿 계약 수

## 실험 1: 무료 체험 범위

가설:

- 무료 실습을 너무 짧게 제공하면 가치가 전달되지 않는다.
- 무료 실습에서 결과 미리보기를 제공하면 회원가입 전환이 높아진다.

변수:

- 1단계만 무료
- 3단계까지 무료
- 결과 리포트 미리보기 제공

측정:

- 회원가입 전환율
- 실습 완료율
- 템플릿 다운로드 의향

## 실험 2: CTA 문구

후보:

- 학습 시작하기
- 무료 실습 시작하기
- 내 문의 응대 기준 만들기
- 고객문의 실습 시작하기

측정:

- CTA 클릭률
- 실습 시작률

## 실험 3: AI 기능 노출 방식

가설:

- AI 자동 생성보다 사용자가 먼저 생각하고 AI가 보조하는 구조가 학습 효과가 높다.

버전:

- 사용자가 분류 후 AI 비교
- AI가 먼저 추천 후 사용자 수정

측정:

- 완료율
- 수정률
- 학습 만족도
- 자신감 변화

## 실험 4: 템플릿 가치

가설:

- 템플릿 다운로드가 유료 전환의 핵심 신호다.

실험:

- 템플릿 일부 공개
- 전체 템플릿 회원가입 후 제공
- 고급 템플릿 유료 제공

측정:

- 회원가입 전환율
- 다운로드율
- 유료 의향

## 실험 5: 업종별 예시

가설:

- 카페, 미용실, 학원처럼 업종 예시를 선택하게 하면 몰입도가 올라간다.

버전:

- 공통 예시
- 업종 선택 후 맞춤 예시

측정:

- 실습 완료율
- 평균 체류 시간
- 만족도

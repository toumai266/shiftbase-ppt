# Initial Founder Package Archive

이 폴더는 Shiftbase 초기 창업 패키지와 과거 PRD 문서를 보존합니다.

아카이브 사유:

- 문서들이 초기 제품 구상, 시장 진입, 기술 아키텍처, 가격 정책까지 넓게 다룹니다.
- 현재 제품 논의는 컨테이너, 페이지형 인터랙티브 학습, 스펙 기반 저작 구조로 좁혀졌습니다.
- 과거 문서 중 일부 용어와 구조는 현재 기준과 맞지 않을 수 있습니다.

현재 기준 문서는 다음 위치를 우선합니다.

- `docs/README.md`
- `docs/common`
- `docs/features`

이 폴더의 문서는 삭제 대상이 아니라 과거 의사결정 참고 자료입니다.

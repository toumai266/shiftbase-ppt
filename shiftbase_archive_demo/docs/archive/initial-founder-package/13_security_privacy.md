# 보안과 개인정보 설계

## 기본 원칙

Shiftbase는 업무 데이터를 다루는 학습 플랫폼입니다. 사용자에게 실제 고객정보를 넣으라고 유도하면 위험합니다. 초기에는 가상 데이터와 예시 데이터를 중심으로 설계합니다.

원칙:

- 실제 고객 개인정보 입력 최소화
- 샘플 데이터 우선 사용
- 민감정보 입력 경고
- AI 결과 발송 전 사용자 검토 필수
- 조직 데이터 분리
- 로그 최소화

## 개인정보 위험 영역

소규모 업체 실습에서 발생할 수 있는 민감정보:

- 고객 이름
- 전화번호
- 주소
- 예약 정보
- 건강 관련 정보
- 결제 정보
- 상담 내용
- 불만 내용

초기 제품은 실제 고객 데이터 업로드 기능을 제공하지 않는 편이 안전합니다.

## 사용자 안내 문구

실습 입력창 주변에 다음과 같은 안내가 필요합니다.

- 실제 고객 이름, 연락처, 주소는 입력하지 마세요.
- 실습에는 예시 데이터를 사용하세요.
- AI가 만든 답변은 발송 전 반드시 직접 확인하세요.
- 업종별 법적 안내가 필요한 경우 전문가 검토가 필요합니다.

## 데이터 저장 정책

저장 대상:

- 사용자 프로필
- 학습 진도
- 실습 제출 결과
- 다운로드 이력
- 결제 내역

주의 대상:

- 사용자가 입력한 문의 내용
- AI 답변 초안
- 상담 기록 예시

권장 방식:

- 입력 전문 저장은 사용자 동의 후 저장
- 기본은 요약 또는 비식별화 데이터 저장
- 기업교육에서는 조직별 정책 선택 가능

## AI 로그 정책

AI 로그는 품질 개선과 비용 추적에 필요합니다. 그러나 민감정보 저장을 피해야 합니다.

저장 권장:

- 기능명
- 모델명
- 토큰 수
- 비용 추정
- 에러 여부
- 입력 요약
- 출력 요약

저장 지양:

- 실제 고객정보가 포함된 입력 전문
- 민감한 상담 내용 전문

## 접근 제어

권한 유형:

- guest
- free_user
- paid_user
- org_user
- org_admin
- admin

원칙:

- 사용자는 자신의 결과만 접근 가능
- 조직 관리자는 자기 조직 구성원 데이터만 접근 가능
- 관리자 접근은 최소 인원만 허용
- 중요한 관리자 작업은 로그를 남김

## RLS 정책 방향

Supabase 사용 시 RLS를 활성화합니다.

정책 예시:

- progress_records는 owner user_id만 select 가능
- practice_submissions는 owner user_id만 select 가능
- organization_members는 같은 organization_id의 org_admin만 조회 가능
- public course와 category는 모든 사용자가 조회 가능
- admin은 service role로 관리 기능 수행

## 파일 보안

자료 파일은 공개 자료와 유료 자료를 분리합니다.

- 무료 템플릿: 공개 링크 가능
- 유료 템플릿: 서명된 URL 사용
- 기업교육 자료: 조직 권한 확인 후 접근

## 결제 보안

- 결제 성공 여부는 클라이언트만 믿지 않음
- 웹훅으로 결제 상태 확정
- order_id 중복 처리 방지
- 환불 상태를 별도 기록

## AI 결과 책임 고지

서비스 내 고지 예시:

AI가 생성한 결과는 업무 보조용 초안입니다. 실제 발송, 계약, 상담, 안내에 사용하기 전 사용자가 직접 검토해야 합니다.

## 업종별 주의

### 클리닉과 의료 관련 업체

- 진단, 치료, 처방으로 오해되는 답변 금지
- 예약 안내와 일반 운영 안내 중심

### 학원

- 학생 개인정보와 성적 정보 입력 주의
- 상담 내용 저장 범위 제한

### 온라인 판매

- 환불, 교환, 배송 정책은 실제 정책 확인 필요
- 소비자 보호 관련 문구 검토 필요

## 보안 체크리스트

MVP 출시 전 확인:

- RLS 적용 여부
- 개인정보 처리방침 작성
- 이용약관 작성
- AI 결과 고지 표시
- 실제 고객정보 입력 경고
- 파일 접근 권한 확인
- 결제 웹훅 검증
- 관리자 접근 권한 제한
- 로그에 민감정보가 저장되지 않는지 확인

## 핵심 판단

Shiftbase의 신뢰는 AI 성능뿐 아니라 안전한 사용 가이드에서 나옵니다. 특히 소규모 업체 대상 제품은 쉽고 안전한 안내가 필수입니다.

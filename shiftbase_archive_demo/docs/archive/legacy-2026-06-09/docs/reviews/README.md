## 산출물 관련 우선 규칙

이 프로젝트에서 산출물은 중요하지만 최상위 목표가 아니다.

산출물은 콘텐츠 품질을 증명하는 장치다. 산출물, 템플릿, 체크리스트, 결과 카드가 많아도 문장 구사력, 콘텐츠 구성력, 기승전결 완결성, 쉬운 학습 경험이 약하면 유료급 콘텐츠로 보지 않는다.

실행자는 다음 순서로 판단한다.

1. 모듈 1개가 기승전결이 있는 챕터인가.
2. 문장 구사력과 콘텐츠 구성력이 상용 도서·강의 수준인가.
3. 학습자가 이해하기 쉽고 긍정적인 학습 경험을 얻는가.
4. 산출물은 위 세 가지를 강화하는가.

산출물은 4번이다. 1~3번을 대신할 수 없다.

## 인터랙티브 관련 우선 규칙

[ai-webbook-global-content-feedback-2026-06-01.md](/C:/Users/vulpe/Desktop/Business/00_AXworkshift/docs/reviews/ai-webbook-global-content-feedback-2026-06-01.md:1)

모든 콘텐츠에 적용되는 최신 피드백이다. 모든 도구와 모든 페이지를 반드시 인터랙티브하게 만들 필요는 없다.

실행자는 다음 원칙을 따른다.

1. 콘텐츠 품질과 이해 용이성이 최우선이다.
2. 인터랙티브, code-canvas, 클릭 요소 수는 품질 근거가 아니다.
3. 단순 매뉴얼, 참고 이미지, 비교표, 예시 문장이 더 선명하면 그것을 선택한다.
4. 인터랙티브는 사용자의 선택, 입력, 분류, 수정, 저장이 실제 판단을 강화할 때만 쓴다.

좋은 매뉴얼 페이지는 나쁜 인터랙티브 페이지보다 낫다.

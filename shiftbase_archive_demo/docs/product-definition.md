# Shiftbase Product Definition

## 한 문장

Shiftbase는 동영상 강의가 아닌 실습형 코스로 학습하는 AI 교육 플랫폼입니다. 사용자는 강의를 보기만 하지 않고, AI가 안내하는 단계형 화면에서 내용을 입력, 선택, 판단하며 여러 가지 AI 활용 상황을 직접 수행합니다.

## 제품의 핵심

Shiftbase는 AI를 설명하는 강의 사이트가 아니다.

사용자가 AI 활용 상황을 화면 안에서 직접 겪게 만드는 코스 플랫폼이다.

기존 온라인 강의가 주로 이렇게 흘렀다면,

```text
강사가 설명한다 -> 사용자가 본다 -> 나중에 혼자 해본다
```

Shiftbase는 이렇게 흘러야 한다.

```text
상황을 만난다 -> 짧게 이해한다 -> 입력한다 -> 선택한다 -> 판단한다 -> AI 결과를 다룬다 -> 다음 상황으로 넘어간다
```

## 사용자에게 보여줄 약속

사용자는 Shiftbase에서 "AI 지식"을 듣기보다 AI를 쓰는 장면을 직접 수행한다.

좋은 코스는 다음 감각을 준다.

- 설명만 들은 것이 아니라 직접 해봤다.
- AI가 만든 결과를 그대로 믿지 않고 비교했다.
- 어떤 선택이 나은지 판단해봤다.
- 비슷한 상황이 오면 다시 시도할 수 있겠다.

## 표현 기준

사용자-facing 문구에서는 아래 표현을 우선 쓴다.

- 실습형 코스
- 단계형 화면
- 직접 입력
- 선택
- 판단
- AI 활용 상황
- 직접 수행
- 결과
- 해보기

아래 표현은 사용자 화면보다 내부 운영 문맥에서만 조심해서 쓴다.

- 학습
- 교육
- 강의
- 수강
- 진도
- 커리큘럼

이전 문서의 낡은 표현은 현재 제품 소개의 중심에 두지 않는다.

## 다루는 AI 활용 상황

초기에는 업무형 AI 활용이 가장 현실적인 시작점이다. 하지만 문서 작성에 갇히면 안 된다.

Shiftbase의 코스 주제는 다음처럼 넓게 잡는다.

- 글, 이메일, 보고, 회의 정리
- 자료 조사와 출처 비교
- 표, 데이터, 숫자 해석
- 이미지와 시각 자료 만들기
- 아이디어 발산과 선택
- 고객 응대와 메시지 톤 조정
- 자동화 흐름 설계
- 여러 AI 도구를 조합하는 작업
- AI 결과의 오류, 누락, 과장 판단

핵심은 특정 결과물 종류가 아니라, 사용자가 AI 활용 상황을 직접 수행하는 것이다.

## 이 제품이 아닌 것

Shiftbase는 아래 방향으로 설명하지 않는다.

- 동영상 강의 플랫폼
- 블로그를 책처럼 정리하는 서비스
- 단순 프롬프트 모음
- AI가 알아서 모든 일을 처리하는 자동화 도구
- 일반 CMS
- 노션 템플릿 판매 서비스

## 좋은 코스의 조건

좋은 Shiftbase 코스는 한 화면 안에서 사용자가 뭔가를 해야 한다.

반드시 복잡한 인터랙션일 필요는 없다. 하지만 사용자는 최소한 다음 중 하나를 해야 한다.

- 입력한다.
- 고른다.
- 비교한다.
- 판단한다.
- 고친다.
- 다음 상황에 적용한다.

설명은 행동을 돕기 위해 존재한다. 설명 자체가 메인이 되면 Shiftbase가 아니다.
